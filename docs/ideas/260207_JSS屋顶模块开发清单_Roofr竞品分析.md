## 📋 背景：为什么要做屋顶模块

### Roofr是谁？

**公司背景：**
- 成立时间：2016年
- 总部：多伦多，加拿大
- 融资情况：Series B，累计融资 $65.4M USD
- 估值：$180M - $200M USD（2026年初）
- 团队规模：150-200人
- 用户量：12,000+ 屋顶公司

**核心产品：**
- 3小时卫星航拍量尺报告（$13-19/份）

- 精美的Proposal生成器
- 在线合同签署
- 支付与融资集成

## 🎯 Roofr的核心强项

### 1. 技术护城河（⭐⭐⭐⭐⭐ 极难复制）

**核心能力：**
- AI + 卫星图自动识别屋顶
- 自动计算面积、坡度（Pitch）、排水沟
- 3D模型渲染

**数据来源：**
- Nearmap / EagleView 高精度航拍
- 成本：$5-10/份数据
- 分辨率：7cm（能看清瓦片纹路）
- 视角：倾斜摄影（东南西北45度）

**复制难度：**
- 需要购买昂贵航拍数据训练AI
- 算法精度需达95%以上
- 需要人工校对团队兜底

**技术架构细节：**
- 前端：Web-First策略，本质是PWA（不是原生App）
- 核心引擎：计算视觉（Computer Vision）+ API驱动
- 后端：Node.js/Python异步架构，AWS/GCP部署
- **关键弱点：** 移动端体验弱，信号差时加载慢，无离线模式

---

### 2. 生态与渠道护城河（⭐⭐⭐⭐ 难）

**关键合作：**
- ABC Supply（北美最大建材供应商）战略投资
- 一键推送订单到供应商仓库
- 深度集成QuickBooks（2025年底）

**粘性来源：**
- "软件+供应链"闭环
- 用户迁移成本高

---

### 3. 金融护城河（⭐⭐⭐ 中等）

**Embedded Finance：**
- 现场贷款/分期服务
- 房主当场申请5年分期
- 极大提高客单价

**复制难度：**
- 技术实现不难
- 难在金融合规和风险控制

---

### 4. 营收数据（⭐⭐⭐⭐⭐ 证明市场存在）

**2026年初估算：**
- 年收入（ARR）：$50M - $80M USD
- 人均产值：~$300K USD/员工
- 用户量：12,000+屋顶公司

**收入结构（混合模型）：**
1. **订阅费（大头）：** $99-169/月/公司
2. **交易费（弹性）：** $13-19/份测量报告
3. **支付手续费（Fintech）：** 从房主→工头支付中抽成

**关键洞察：**
- 从"卖报告"变成"管生意"实现营收爆发
- 高频带低频：CRM锁住用户，报告引流
- 供应链杠杆：ABC Supply投资后直接从材料流水分钱

## ❌ Roofr的关键弱点（JSS的机会）

### 1. "管杀不管埋"

**痛点：**
- 强项在签单前（测量、报价、合同）
- 签单后（施工、进度、现场管理）功能薄弱
- 没有收据管理、材料成本追踪

**JSS机会：**
- 专注"签单后"的利润守护
- LedgerSnap财务闭环

---

### 2. 移动端体验差

**痛点：**
- 本质是PWA（网页套壳），不是原生App
- 工地信号差时加载慢
- 无法离线查看图纸
- 拍照功能极弱（一张一张传）

**JSS机会：**
- 原生App流畅体验
- 离线模式
- 批量连拍快门

---

### 3. "树遮挡"精度问题

**痛点：**
- 温哥华树木茂盛（西区、北温）
- 卫星图看不清屋檐线
- 坡度误差可达10-15%

**JSS机会：**
- 人机协作模式
- 师傅现场核实100%准确

---

### 4. 收费"连环套"

**痛点：**
- 月费 + 报告费 + 短信费
- 忙碌公司月均开销 $500-800
- 小工头负担重

**JSS机会：**
- 纯订阅模式
- 更透明定价

---

### 5. 缺乏"眼缘"与信任温度

**痛点：**
- 冷冰冰的科技报告
- 试图用AI取代师傅判断
- 让师傅像"中介"而非"专家"

**JSS机会：**
- 用数据支撑师傅专业
- 透明化施工证据链
- 增强人与人信任

**核心洞察（来自20年经验）：**
> "房主找contractor还是喜欢看眼缘的，那种人与人之间的交流建立信任。不是几份漂亮报告可以代替的。"

**两种获客逻辑对比：**

| 维度 | Roofr "科技外壳"模式 | JSS "专业邻居"模式 |
|------|---------------------|-------------------|
| 第一步 | 发精美卫星图报告（可能出错） | 师傅上门，亲自爬梯子，建立眼缘 |
| 信任来源 | 看起来很贵的PDF | 真实现场照片、精准核实、坦诚交流 |
| 沟通重点 | "我的系统说你家屋顶是30 Squares" | "我刚上去看了，烟囱闪缝确实烂了，我拍了照，你看" |
| 房东感受 | 这公司很有钱，规模很大 | 这老板懂行，做事踏实，我放心 |

**JSS的"信任增强"功能：**
1. **当面核实的仪式感：** 师傅带房东看App实时照片
2. **透明施工证据链：** 每个步骤拍照，房东手机实时看进度
3. **"有理有据"的加项：** 现场拍照给房东看，不伤眼缘还能多挣钱

## 🚀 JSS屋顶模块核心功能

### Phase 1：MVP核心功能

#### 1️⃣ SnapDraft（廉价底图 + 人工核实）

**功能描述：**
- 用户输入地址
- 调用便宜卫星API（Google Maps / Mapbox）
- 自动识别屋顶大致轮廓（开源库即可）

**成本优势：**
- Google Maps：$0（每月$200免费额度）
- vs Roofr：$5-10/份

**工作流：**
```
1. 远程：输入地址 → 自动拉取卫星图 → AI识别大致轮廓
2. 现场：师傅上梯子 → 核实尺寸 → 确认细节
3. 生成：自动生成材料单和报价
```

---

#### 2️⃣ SnapPitch（感应测坡度）

**功能描述：**
- 利用手机陀螺仪（Gyroscope）
- 师傅把手机贴在屋顶面
- 1秒读取精准Pitch（坡度）

**技术实现：**
- PWA可调用陀螺仪API
- 原生App体验更好

**公式：**
```
实际面积 = 投影面积 × √(1 + (Pitch/12)²)
```

**价值：**
- 比AI猜的准得多
- 完全免费（无API成本）
- 100%精确

---

#### 3️⃣ SnapEvidence（原生存证相机）

**功能描述：**
- 原生App批量连拍
- 离线模式（地下室无信号照样拍）
- 自动打水印（GPS、时间、工单号、天气）

**核心优势：**
```
PWA拍照痛点：
- 一张一张传
- 断网打不开
- 加载慢、卡顿

JSS原生App：
- 连拍模式（咔咔咔）
- 离线缓存
- 有网自动异步上传
```

**法律级存证：**
- BC省小额法庭（CRT）认可
- 带GPS/时间戳的照片
- 防赖账、保险理赔

---

#### 4️⃣ SnapOrder（极简材料单）

**功能描述：**
- 核实完尺寸和坡度
- 一键生成材料清单

**本地化：**
- 预置温哥华建材商（Roofmart、IKO）
- 瓦片规格、每包覆盖面积
- 自动计算BC省回收费（Eco Fees）

**输出：**
- 瓦片数量
- 钉子
- 防水层
- 闪缝（Flashing）
- 损耗率自动计算

---

### Phase 2：差异化功能

#### 5️⃣ SnapHistory（QR冰箱贴 / 房屋数字身份证）

**核心创意：**
- 完工生成专属QR码
- 打印成冰箱贴留在房子里
- 房子卖了，新屋主扫码看历史

**记录内容：**
1. **施工历史：**
   - Before/After照片
   - 材料清单（瓦片品牌、颜色）
   - 施工日期

2. **隐蔽工程：**
   - 封板前的电线走线
   - 水管位置照片
   - 防水层细节

3. **质保手册：**
   - 50年质保书
   - 材料发票
   - 维护建议

**商业价值：**
- 永不丢失的活名片
- 房子帮你找客户
- 新屋主第一个联系你

**深度价值分析：**

1. **长效营销（跨越交易）：**
   - 传统名片：干完活就被扔了
   - QR冰箱贴：粘在厨房核心位置
   - **关键：** 房子卖了，新屋主进门第一眼看到

2. **"房屋病历本"概念：**
   - 不只是照片，是房子的完整档案
   - 结构隐蔽工程：封板前电线走线、水管位置
   - 材料清单：墙漆色号、地板品牌、瓷砖型号
   - 质保手册：50年质保书、电器发票

3. **自动转化新客户：**
   - 新屋主想翻新时，第一个联系"了解这房子历史"的人
   - 不需要找客户，房子会帮你找客户
   - 建立温哥华"房屋数据库"

**实现流程：**
```
1. 项目完工 → JSS点击"完工归档"
2. 自动生成专属网页 → 精华照片 + 材料清单 + 维护建议
3. 生成QR码 → 本地工厂批量定制冰箱贴（带JSS标志）
4. 留在房子里 → 永久营销资产
```

**这种"眼缘"和"情怀"是Roofr永远学不会的。**

---

#### 6️⃣ 业主实时看板

**功能描述：**
- 业主在线查看施工进度
- 实时照片推送
- 透明度建立信任

**工作流：**
```
师傅：拍照 → 自动上传
业主：手机收到通知 → 查看进度
```

**价值：**
- 解决"看不到偷工减料"的担忧
- 比精美PDF更有说服力
- 可分享到朋友圈（免费广告）

---

#### 7️⃣ 一键加项（Change Order）

**场景：**
- 施工中发现底板烂了

**流程：**
```
1. 师傅拍照 → 圈出烂的地方
2. 输入"需要加$500"
3. 点击发送
4. 业主手机收到审批请求
5. 点击同意 → JSS自动更新合同
```

**价值：**
- 现场即时沟通
- 避免事后扯皮
- 有理有据，不伤眼缘

## 💬 核心洞察总结

### Roofer的真正痛点是什么？

**成熟屋顶老板的核心问题：**
> "如何保住我的利润率（Margin）？"

**痛点分层分析：**

| 痛点类别 | Roofr解决了什么？ | JSS应该解决什么？ |
|---------|------------------|------------------|
| **成交** | 漂亮PDF、快速报价、分期付款 | 实打实现场证据，让房东没法赖账 |
| **准确性** | 卫星估算（适合拍脑袋） | 现场核实（AR/感应器），材料不多不少 |
| **沟通** | 办公室与房东合同往来 | 工地与办公室实时同步（原生App） |
| **财务** | 软件订阅和支付 | 每一分钱收据的归档（LedgerSnap） |

**利润流失的真实场景：**
1. **"估算"≠"实战"：** AI算的料，开工才发现要加10张底板
2. **现场证据缺失：** 垃圾清理、闪缝更换无照片，房东赖账
3. **离线折磨：** 工地信号差，PWA就是砖头
4. **材料算错：** 7/12和9/12坡度误差，损失数千加币

**关键判断：**
> Roofr解决的是"把蚊子吸引过来（签单）"，但没能"把蚊子拍死（高效完工并收回款项）"。

## 📌 关键战略决策记录

### 决策1：不硬碰硬复制Roofr的卫星测量

**理由：**
- Nearmap/EagleView数据成本：$5-10/份
- 需要昂贵的AI训练和人工校对团队
- 复制难度：⭐⭐⭐⭐⭐（极难）

**JSS策略：**
- 用Google Maps免费底图（$0成本）
- 师傅现场核实确保100%准确
- 把Roofr的95%变成JSS的100%

---

### 决策2：必须做原生App，不能只做PWA

**Roofr的弱点：**
- 本质是PWA（网页套壳）
- 工地信号差时体验极差
- 拍照功能弱（一张一张传）
- 导致用户还要花$30+/月买CompanyCam

**JSS优势：**
- 原生App批量连拍
- 离线缓存（地下室无信号照样拍）
- 自动水印（GPS、时间、工单号）
- 法律级存证（BC省小额法庭认可）

---

### 决策3：专注"签单后"，不抢"签单前"

**市场定位：**
- Roofr：销售工具（Pre-sale）
- JSS：利润守护者（Post-sale）

**差异化：**
| Roofr | JSS |
|-------|-----|
| 办公室里算命（估算） | 工地上定乾坤（实测） |
| 签单前（Measurement → Proposal → Contract） | 签单后（Execution → Evidence → Payment） |
| 让师傅像"中介" | 让师傅更像"专家" |

---

### 决策4：QR冰箱贴是长期竞争优势

**不可复制性：**
- 人情味和本地化深度
- 房屋历史档案（隐蔽工程、材料规格）
- 跨交易营销（房子卖了还能带来新客户）

**与LedgerSnap联动：**
- 完整的"房屋生命周期数据"
- 从施工到财务的完整闭环
