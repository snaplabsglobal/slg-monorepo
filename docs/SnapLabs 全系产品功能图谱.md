## SnapLabs 全系产品功能图谱

### 1. LedgerSnap (LS) —— 效率切入点

**定位**：针对个体承包商与工头的“刺刀级”记账工具。核心解决“收据多、报税难、成本乱”的痛点。

- **极速影像采集 (Snap)**：0 延迟原生相机调用，支持离线拍摄与背景异步同步（Offline-First）。

- **AI 智能双轨解析**：
  
  - **核心 6+3 字段**：自动提取商户、日期、总额、税金、类别、项目等关键核账信息。
  
  - **后台物料解析**：在用户不可见的情况下，自动解析并存储 **Line Items (物料明细)** 到 JSONB 字段，为升级 JSS 预留数据。

- **商户别名进化 (Vendor Aliases)**：L1-L3 三级学习机制，自动将“H.D.”纠偏为“Home Depot”，实现一人纠偏，全网受益。

- **税务与财务对冲**：支持自动税务拆分，生成符合会计准则的一键导出报表。

- **乐观交互 (Optimistic UI)**：用户确认时秒回馈，无需等待服务器响应。

### 2. JobSite Snap (JSS) —— 管理制高点

**定位**：针对中大型建筑公司的项目全生命周期管控平台。

- **物料级成本核算 (Line Item Tracking)**：将 LS 隐藏的明细数据视觉化，追踪每一根木材、每一颗螺丝的流向。

- **多员协作矩阵**：
  
  - **员工上传端 (Permanent Free)**：支持无限量工人拍照上传，自带 GPS 与语音备注。
  
  - **管理权限隔离**：基于 RBAC 架构，确保老板看总账，财务看明细，工人仅限上传。

- **项目动态看板**：实时显示各个工地的预算执行情况（Budget vs Actual），自动预警超支风险。

- **员工效能统计**：量化分析各成员的上传频率与准确度，辅助管理决策。

### 3. SnapLabs Global (SLG) —— 协议底层

**定位**：建筑行业的数字化基础设施与 AI 协议中心。

- **三表联动多租户架构**：底层支持一个用户在不同组织、不同角色（Owner/Worker/Admin）间无缝切换。

- **Snap-LLM 行业模型**：基于海量建筑行业收据训练的垂直领域模型，确立行业识别标准。

- **跨平台 Design System**：统一的“石墨黑+冰川白”高端 UI 视觉语言，适配 Web 与移动端。

## 📊 产品功能梯度对比

| **功能特性** | **LedgerSnap (LS)** | **JobSite Snap (JSS)** |
| -------- | ------------------- | ---------------------- |
| **收据识别** | 全量识别，展示核心字段         | 全量识别，展示 **物料明细**       |
| **项目管理** | 基础关联（仅作标签）          | 深度管控（预算、周期、看板）         |
| **协作名额** | 仅限个人或 1-2 名帮手       | **无限制** 员工协作           |
| **数据深度** | 财务会计维度              | 经营管理维度                 |
| **商业逻辑** | 提高**效率**，节约报税时间     | 提高**利润**，掌控项目流向        |
