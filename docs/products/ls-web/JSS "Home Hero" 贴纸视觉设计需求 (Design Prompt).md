### JSS "Home Hero" 贴纸视觉设计需求 (Design Prompt)

老板，这张贴纸就是咱们的**“门面”**。一定要让师傅觉得贴在锅炉上不丢人，甚至觉得很有面子。

#### 🎨 视觉规范 (Visual Identity)

- **主色调**：**Architectural Blue (#1A4F8B)** 作为底色或边框，代表专业与结构。

- **点缀色**：**Vibrant Orange (#FF7E21)** 用于“Scan Now”或紧急联系按钮，代表行动力。

- **材质要求**：必须是 **High-Tack Vinyl (高粘性乙烯基)**，防水、防油、耐温高达 100°C。

#### 📐 布局描述 (Layout Prompt for Designer/AI)

1. **顶部 (Header)**: `SERVICE RECORD` (加粗大字，深蓝色)。

2. **中间左侧 (QR Code)**: 一个醒目的大二维码，四周留白。

3. **中间右侧 (Call to Action)**:
   
   - 文字：`SCAN TO SEE HISTORY`
   
   - 下方带一个橙色的小箭头。

4. **底部 (Footer)**:
   
   - 左侧：由 **JobSite Snap** 提供技术支持。
   
   - 右侧：空白区域供师傅用油性笔签名字（或系统激活后自动关联）。

5. **侧边 (Side)**: 留出位置印上合作批发商的 Logo（如：*Partnered with Andrew Sheret*）。
